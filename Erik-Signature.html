How to put it in Outlook (web / New Outlook):
1. Click Copy my signature above.
2. In Outlook, click the gear ⚙ (Settings) → Accounts → Signatures (or type “signature” in the settings search).
3. Click + New signature, name it “Erik 2026”, click in the big box, and press Ctrl + V to paste.
4. Set it as the default for new messages (and replies if you like). Save.
