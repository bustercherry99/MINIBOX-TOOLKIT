A copy and paste toolkit for rendering the 15 second MiniBox+ product film in an AI video engine (Veo 3 via Gemini, Higgsfield, Kling, Runway). Every prompt is anchored on the real product photo so the device stays true to life. Read the workflow note first, it will save you credits and headaches.

THE HYBRID WORKFLOW (read this). AI video is incredible at glossy exterior beats: the spin, the floating handpiece, the tablet ignition. It is unreliable at the exploded internals, it does not know your real 16.3 L canister and will invent parts. So: render the exterior hero shots in the AI tool using the prompts below, render the exploded internals beat from the 3D Showcase file we already built, then cut the pieces together in any editor (CapCut is free and fast). One clean 15 second film, best of both.

Tool Cheat Sheet

Veo 3 · via Gemini / Flow
best all-rounder + native audio
  • 8 second clips, so Shots 1 and (2+3) can each be one generation.
  • Use Frames to Video: upload the real MiniBox photo as the start frame to lock fidelity.
  • It can speak the VO, paste the line in quotes inside the prompt.
  • 16:9, request "no on screen text" so it does not add captions.
Higgsfield
best cinematic camera control
  • Image to video from the real photo, then apply a camera preset: Orbit for Shot 1, Push In for Shot 3.
  • Great for the weightless float, use slow motion strength.
  • Stitch its 5s clips for the exterior beats.
Kling 2.1 / Runway Gen-4
strong image-to-video backups
  • Start frame = real photo. Kling supports start AND end frame for tight control.
  • 5 to 10s clips, keep motion "smooth / cinematic".
  • Good fallback if Veo access is limited.
Consistency Rules
follow these on every shot
  • Always feed the same real product photo as the reference frame.
  • Keep the master style block identical, word for word.
  • Lock the seed if the tool allows it.
  • Render 2 or 3 takes per shot, pick the cleanest, that is normal.
