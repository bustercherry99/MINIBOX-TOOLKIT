“What’s a Z-score?” — the question you’ll get

How many steps a value sits above or below the average healthy person of the same age, height, sex and ethnicity. 0 means exactly predicted. Anything below −1.645 counts as abnormal — that’s the LLN line. A −4 means “far, far below normal.”

It’s the modern ATS/ERS standard because one number tells you how far off a patient is, no matter which parameter you’re looking at — and unlike “80% of predicted,” it already accounts for the fact that normal ranges differ by age and size.

If the report has no Z-score column, use the % Predicted column instead and the Composer falls back to the 80% rule. The z-score is better where you have it.
