MiniBox+ Master Toolkit v3.0 — Offline Copy
============================================

TO USE:  Unzip this whole folder anywhere, then double-click  index.html

IMPORTANT: Keep these together in one folder (don't email index.html by itself):
  - index.html            (the toolkit)
  - materials-data.js     (the PDFs baked in — this is what makes real
                           email attachments work, even with no internet)
  - materials\            (the same PDFs as normal files, for View/Download)

WHAT'S NEW IN v3.0
  - Emails now carry REAL attachments. "Outlook Draft" downloads a
    ready-to-send draft with the selected PDFs already inside —
    double-click it, review, hit Send.
  - On New Outlook / Outlook Web, use the "Outlook Web + files" button:
    the email opens written and the PDFs download alongside so you drag
    them in. The toolkit never writes "attached" unless files are
    actually coming along.
  - Materials Library (Forms tab): every official PDF — view, download,
    or email any of them in two clicks.
  - My Rep Profile now asks WHICH Outlook you use and highlights your
    best send button everywhere.

UPDATING A PDF LATER
  Replace the file in materials\ (same name), then right-click
  build-materials-data.ps1 → Run with PowerShell. Done.
