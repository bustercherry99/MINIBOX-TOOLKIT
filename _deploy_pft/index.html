i About this tool

A living helper. Codes that can't be billed together (bundled or mutually exclusive under NCCI) grey out the moment you select the one that controls them, so you only ever build a valid combination. Each payer's covered-diagnosis list and frequency rules vary and are not hard-coded — the tool routes you to that payer's own policy to confirm before you submit.
