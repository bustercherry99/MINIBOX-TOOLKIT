WHAT THIS FOLDER IS
===================
Old clutter cleared out of the MiniBox Sales Toolkit folder on 2026-07-15.
Nothing here is live or in use. It's kept only as a safety net.

Two kinds of things are in here:

  old-versions/  = superseded numbered versions of documents. For every
                   document family we kept the two NEWEST versions in the
                   main folder and moved the older ones here.

  backups/       = automatic ".bak" editor snapshots (point-in-time saves
                   made before edits). The current live files stayed put.

The live files that are still in the main folder and _deploy_toolkit are
untouched and working.

WHEN YOU'RE READY
=================
Once you've confirmed nothing's missing, you (or Claude next time) can
delete this whole "_OLD_versions" folder to reclaim ~36 MB. Safe to delete.
