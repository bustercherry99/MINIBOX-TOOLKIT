PULMSUITE LIVE DEMO, DEPLOY INSTRUCTIONS
=========================================

This folder is a ready-to-deploy standalone site. One file: index.html
(a copy of PulmSuite_Interactive_v1.6).

FASTEST PATH (2 minutes, no command line):
1. Go to vercel.com and log in
2. Click "Add New" then "Project", or just go to vercel.com/new
3. Drag this whole PulmSuite_Demo_Deploy folder onto the page
4. Name the project something clean, for example: pulmsuite-demo
5. Click Deploy

You get a URL like https://pulmsuite-demo.vercel.app

OPTIONAL: in the Vercel project settings you can add a custom domain
later (for example demo.pulmsuite.com) without redeploying anything.

Once it is live, send Claude the URL and the animate-me button and QR
code go into the one-pager and two-pager PDFs pointed at it.
